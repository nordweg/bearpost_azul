## Installation
Add this line to your application's Gemfile:

```ruby
gem 'bearpost_azul'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install bearpost_azul
```

Generate files
``` bash
# rails generate bearpost_azul
rails generate bearpost_azul:install
```
